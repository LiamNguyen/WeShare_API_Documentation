API Documentation

Authentication

User sign in

Method:

POST

/user/login

Headers:

  • Content-Type: application/json

    • Request Body:

    {

    "userLoginId": "demoUsername",
    "userPassword": "demoPassword"

    }

    Success: (200)

    {

    "userId": "13",
    "sessionToken": "hhq88unk9lrcvjoq147bb5mh0s"

    }

    Error: User not found (404)

    {

    "documentation": "http://localhost:8080/api-weshare/webapi/user/register",
    "message": "Seems like you haven't registered. Please register as a new user with the provided URL in documentation",
    "statusCode": 404

    }

    Error: Invalid user or password (403)

    {

    "documentation": "http://localhost:8080/api-weshare/webapi/user/register",
    "message": "Invalid username or password",
    "statusCode": 403

    }

    Profile

    Get profile

    Method:

    GET

    /user/profile

    Headers:

  • Content-Type: application/json
  • Authorization: [Session Token] - Get token from login or register

    • Success: (200)

    {

    "createdAt": "2017-02-24T10:25:07.919",
    "userEmail": "phuc@gmail.com",
    "userFirstName": "Thien Phuc"
    "userLastName": "Nguyen"
    "userID": "4"
    "userPhone": "068452597"
    "userTwitter": "phuc@gmail.com"

    }

    Error: Unauthorized (401)

    {

    "documentation": "http://weshare-api.documentation.com",
    "message": "Invalid authorization token. Please login first to get session token",
    "statusCode": "401"

    }
